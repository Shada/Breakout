Map standard

filformat: bmp

Antal pixlar:	bredd x höjd
2D		150 x 100
Scroll		150 x 300
Cylinder	300 x 100
Sfär		300 x 100


Första punkten innehåller info om banan:
R = 	lvlnr  (Ordningen de skall spelas i)
G = 	Typ av bana (tema)
		0 = testing
		1 = Wind
		2 = Water
		3 = Fire
		4 = Earth 
B = 	Svårighet (Hur ofta random event ska ske. 1=simpel, 10=svår)

(Eftersom den första punkten ligger i ett hörn så kommer vi inte sätta något objekt där och hoppa över den när man läser in objekten.)

Resterande punkter innehåller vad som finns i området.
R = 	Typ av object:
		0 och 255 = Inget
		12 = Pad startposition
		24 = Bollens startposition
		36 = Brick (1 liv)
		48 = Brick (2 liv)
		60 = Brick (3 liv)
		72 = Brick (Oförstörbar)
		84 = Speed increase
		96 = Speed decrease
		108 = Padsize increase
		120 = Padsize decrease
		132 = Inverted controlls
		144 = Rotationspeed increase
		156 = Rotationspeed decrease
		168 = Extra liv
G =	Vilken texture objectet ska ha. (T.ex. viken typ en brick är)
B =	Vilken model som objectet ska använda.
